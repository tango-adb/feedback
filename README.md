# Tango Web App

Tango is an Android management tool running in Web browsers. Currently it runs on Chromium-based browsers, including Google Chrome, Microsoft Edge, Opera, etc. It works on PC and Android, but not iOS.

## Features

-   Device Overview
    -   Connected WiFi network name, IP address, battery level, etc.
-   File Management
    -   List files and folders
    -   Upload to device
    -   Download from device
    -   Rename
    -   Delete
-   App Management
    -   List installed Apps
    -   Install APK
    -   Uninstall apps
    -   Backup/restore app data
-   Screen Mirroring
    -   Mirror device screen in low latency
    -   Control device with mouse, touch and keyboard
    -   Mirror device camera (requires Android 12)
    -   Mirror individual apps (can still use your phone normally, requires Android 12)
    -   (Work in progress) Audio forwarding (requires Android 11)
    -   (Work in progress) Recording video
-   Other Advanced Features
    -   Dump device logs
    -   Collect device diagnostics information
    -   Toggle ADB over WiFi
    -   Power menu
